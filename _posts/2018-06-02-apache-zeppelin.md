## Apache Zeppelin Install
- [홈페이지](https://zeppelin.apache.org/) 접속 후 다운로드
	- 용량이 큰 것은 빅쿼리, 카산드라 등이 모두 빌드된 것이고 작은 것은 스파크만 빌드
- 압축 풀기
- ```cd /folder``` : 압축을 푼 폴더로 들어가기
- 실행 : ```./bin/zeppelin.sh```
- localhost:8080
